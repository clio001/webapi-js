The data underlying this web application is provided by The New York Times. A variety of web APIs are offered to access dedicated content and meta data for web developer through the website's Developer Network. For an overview of the APIs available:

  • Archive API
  • Article Search API
  • Books API
  • Community API
  • Most Popular API
  • Movie Reviews API
  • RSS Feeds
  • Semantic API
  • Times Tags API
  • Times Wire API
  • Top Stories API

The Icy News web application uses the Article Search API and draws on a series of query and filter options provided by the API. It combines the query, publication year, and the geolocation filter options to create a user-generated, dynamic request.

The meta data provided as part of the Article Search API JSON offers various options to further filter the results returned from the user-generated API request. The Icy News web application enables users to filter results by news desk, byline, month, and date.
